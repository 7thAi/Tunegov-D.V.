# WPF-приложение «Управление пользователями»

## Описание проекта

Данный проект представляет собой настольное WPF-приложение, разработанное на языке программирования C# с использованием технологии Entity Framework и СУБД Microsoft SQL Server. Приложение предназначено для автоматизации процессов регистрации, авторизации и управления учетными записями пользователей с разграничением прав доступа по ролям.

В рамках практики реализованы механизмы защиты данных, проверки корректности ввода, работа с базой данных и визуализация данных в виде таблиц и плиток с фотографиями пользователей.

---

## Функциональные возможности

### Гость
- Регистрация нового пользователя;
- Авторизация в системе.

### Пользователь
- Просмотр списка пользователей в виде плиток;
- Поиск пользователей по ФИО;
- Фильтрация пользователей по роли;
- Сортировка списка пользователей;
- Просмотр фотографий пользователей;
- Выход из приложения.

### Администратор
- Все возможности пользователя;
- Добавление новых пользователей;
- Редактирование данных пользователей (ФИО, логин, роль, пароль);
- Удаление пользователей;
- Загрузка и изменение фотографии пользователя.

---

## Используемые технологии

- Язык программирования: **C#**
- Платформа: **.NET Framework**
- Технология интерфейса: **WPF**
- Работа с базой данных: **Entity Framework**
- СУБД: **Microsoft SQL Server**
- Хеширование паролей: **SHA-1**
- UML-моделирование: **PlantUML**
- Система контроля версий: **Git**

---

## Структура базы данных

Основной таблицей базы данных является таблица `User`, содержащая следующие поля:

- `ID` — уникальный идентификатор пользователя;
- `FIO` — фамилия, имя и отчество пользователя;
- `Login` — логин пользователя;
- `Password` — хэш пароля пользователя;
- `Role` — роль пользователя (Администратор / Пользователь);
- `Photo` — путь к фотографии пользователя.

---

## Особенности реализации

- Реализовано разграничение прав доступа по ролям;
- Пароли пользователей хранятся в базе данных в зашифрованном виде;
- Реализованы проверки корректности вводимых данных;
- Используются всплывающие подсказки (ToolTip);
- Реализованы ограничения на ввод символов;
- При отсутствии фотографии пользователя отображается изображение по умолчанию;
- Интерфейс автоматически адаптируется к изменению размеров окна.
